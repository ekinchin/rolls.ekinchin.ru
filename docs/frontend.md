# Frontend

Визуальная идея сайта - фотопленки как вертикальные колонки. На главной странице одна колонка соответствует одному roll и напоминает сушащуюся пленку.

## Главная галерея

Файлы:

- `src/layouts/gallery.njk`
- `src/styles/gallery.css`

Главная галерея устроена как горизонтальная лента:

- `.gallery_container` скроллится по горизонтали;
- `.roll_container` скроллится по вертикали независимо от соседних колонок;
- `scroll-snap-type: x mandatory` помогает фиксировать колонки при горизонтальной прокрутке;
- `overscroll-behavior-y: contain` удерживает вертикальный скролл внутри текущей колонки.

Важный контракт: не убирать `overflow-y: auto` у `.roll_container`, иначе ломается дизайн независимых висящих пленок.

## Страница roll

Файлы:

- `src/layouts/contacts.njk`
- `src/styles/film.css`

Страница roll состоит из описания пленки и адаптивной сетки фотографий:

```css
grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
```

Это позволяет сетке естественно подстраиваться под ширину экрана без большого набора breakpoint-ов.

## Lightbox

Файлы:

- `src/layouts/contacts.njk`
- `src/styles/film.css`
- `src/scripts/lightbox.js`

Lightbox открывает фото внутри страницы вместо прямого перехода на JPG.

Подход:

- ссылки на изображения остаются обычными `<a href="/rolls/.../photos/1.jpg">`;
- JS ищет ссылки по `data-gallery-link`;
- при клике JS отменяет стандартный переход и открывает overlay;
- без JS ссылка работает как fallback;
- навигация поддерживает кнопки, `ArrowLeft`, `ArrowRight`;
- закрытие поддерживает `Escape`, кнопку close и клик по серому фону.

Фон просмотрщика нейтрально-серый. Это сделано намеренно, чтобы не конкурировать с фотографиями и не менять восприятие контраста слишком агрессивно.

## CSS-подход

- Общие переменные лежат в `src/styles/variables.css`.
- Базовые стили страницы - `style.css`.
- Главная галерея - `gallery.css`.
- Страница roll и lightbox - `film.css`.

Стили подключаются в `base.njk`, а файлы копируются в `dist` через passthrough в Eleventy.
