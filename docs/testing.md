# Проверки

Главная команда:

```sh
npm test
```

Она запускает shell-тесты из `test/`.

## Тесты

### `test/add-new-roll.test.sh`

Проверяет генератор нового roll на временной копии проекта:

- запуск без аргументов;
- вычисление следующего номера;
- установку даты публикации;
- генерацию front matter;
- создание `xs-*` и рабочих JPG;
- совпадение количества `photos` с фактическими исходными файлами.

### `test/layout-css.test.sh`

Фиксирует ключевые CSS-контракты главной галереи:

- горизонтальный скролл общего контейнера;
- вертикальный скролл отдельных roll-колонок;
- scroll snap;
- удержание вертикального overscroll внутри колонки.

Этот тест появился из регрессии: вертикальный скролл `.roll_container` был случайно удален, хотя он является частью основной идеи дизайна.

### `test/lightbox-ui.test.sh`

Проверяет, что lightbox не выпал из шаблонов и сборки:

- есть `data-gallery-link`;
- есть lightbox-разметка;
- есть кнопки prev/next;
- подключен `/scripts/lightbox.js`;
- `src/scripts` копируется Eleventy;
- JS содержит клавиатурную навигацию.

## Сборка

```sh
npm run build
```

Команда должна успешно собрать сайт в `dist`.

Перед публикацией стоит запускать:

```sh
npm test
npm run build
```

## Ручная проверка

После изменений frontend-а проверить в браузере:

- главная страница горизонтально скроллится между roll;
- каждая roll-колонка вертикально скроллится независимо;
- страница roll открывает фото в lightbox;
- стрелки влево/вправо меняют фото;
- `Escape` закрывает просмотрщик;
- при отключенном JS ссылка на фото открывает JPG напрямую.
