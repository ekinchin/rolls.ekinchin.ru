# Документация проекта

Этот каталог описывает технические решения сайта `rolls.ekinchin.ru`: как устроена сборка, как добавляются новые фотопленки, какие frontend-подходы уже приняты и чем проверять изменения.

## Содержание

- [Архитектура](architecture.md) - структура проекта, Eleventy-конфигурация, данные и шаблоны.
- [Добавление нового roll](content-workflow.md) - workflow подготовки нового поста и работа `add-new-roll.sh`.
- [Frontend](frontend.md) - принципы верстки главной галереи, страницы roll и просмотрщика фото.
- [Проверки](testing.md) - тестовые сценарии и команды перед публикацией.

## Основные команды

```sh
npm run dev
npm run build
npm test
npm run new
```

`npm run new` создает новый пост из каталога `new-roll`. Перед запуском туда должны быть положены исходные JPG и заполнен или подготовлен `new-roll/index.md`.
